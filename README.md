
# Documentador — Next.js + TypeScript + Tailwind + CSS Modules

Cambios pedidos:
- **Sin slide-over** para el formulario: el Documentador vive en /aceleradores dentro del contenido principal.
- **Video de fondo** (GFT) en el hero.
- **Logos**: GFT (SVG local) y Banco de Occidente (URL).
- **Iconos** provistos (acelerador, cargar-archivo, aprobación, documentador), sin emojis.
- **Transacción única**: se envía `fileBase64` + `projectName` y se descarga el DOCX de respuesta (placeholder).

## Ejecutar
```bash
npm install
npm run dev
```
